# Marketing Agency Landing Page

Self-contained static landing page implementation.

## Preview locally

```bash
python -m http.server 8000 --bind 0.0.0.0
```

Then open:

- `http://127.0.0.1:8000`

## Files

- `index.html` – page structure
- `styles.scss` – source SCSS with variables and section styling
- `styles.css` – runtime stylesheet
- `script.js` – smooth-scroll and reveal animations
- `assets/*.svg` – local artwork so previews work without external image access
